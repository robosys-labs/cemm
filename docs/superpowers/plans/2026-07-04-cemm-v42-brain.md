# CEMM v4.2 Semantic Brain — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use subagent-driven-development (recommended) or executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transform CEMM from its dual-graph seed runtime (`Pipeline` deciding between `SemanticEventGraph` and `UOLGraph` paths) into the single-authority `SemanticKernelRuntime` architecture where `UOLGraph` is the sole working graph, `GraphPatch` is the only durable-write mechanism, and all 5 architecture-specified durable types exist as first-class citizens.

**Architecture:** CPU/Atom/Brain metaphor — `SemanticKernelRuntime` is the CPU, `UOLGraph` is the register file (volatile working memory), `SemanticAttentionController` is the instruction scheduler/focus selector, `PatchValidator` is the memory-management unit gating all writes to durable storage, and the 5 durable types (`ConceptAtom`, `OperationalPort`, `PredicateSchema`, `CausalAffordance`, `ConstructionAtom`) are the cache-line structures the brain persists.

**Tech Stack:** Python 3.13, dataclasses, existing CEMM types (UOLGraph, GraphPatch, UOLAtom, UOLEdge), existing stores (ClaimStore, ConceptLattice, ConstructionLattice), existing operators (RememberOperator, AnswerOperator)

**Key breakthrough:** The current code has ALL the right components but the wrong wiring. `Pipeline.run()` (664 lines) mixes legacy + modern paths with conditionals. The fix is not to rewrite components — it's to define the clean interfaces (`SemanticKernelRuntime`, `RuntimeCycleResult`, `PatchValidator`, 5 durable types) and re-wire existing components behind them. This eliminates 6+ conditional branches, removes backward-compat properties from `UOLGraph`, and makes `GraphPatch` the sole durable-write mechanism.

@dataclass
class SemanticFingerprint:
    key_tokens: set[str] = field(default_factory=set)
    surface_tokens: set[str] = field(default_factory=set)
    role_tokens: set[str] = field(default_factory=set)

    def jaccard(self, other: SemanticFingerprint) -> float:
        union = self.key_tokens | other.key_tokens | self.surface_tokens | other.surface_tokens | self.role_tokens | other.role_tokens
        if not union:
            return 0.0
        intersection = self.key_tokens & other.key_tokens | self.surface_tokens & other.surface_tokens | self.role_tokens & other.role_tokens
        return len(intersection) / len(union)

```python
class SemanticKernelRuntime:
    def __init__(self, store, registry):
        self._store = store
        self._perceptor = MeaningPerceptor(store)
        self._graph_builder = MeaningGraphBuilder(...)
        self._attention = SemanticAttentionController()
        self._retrieval_executor = RetrievalExecutor(store)
        self._resolver = RuntimeResolver(...)
        self._planner = ActResolutionPlanner()
        self._patch_extractor = GraphPatchExtractor()
        self._patch_validator = PatchValidator(store)
        self._consolidator = ConceptConsolidator(store)

    def run_turn(self, signal: Signal, kernel: ContextKernel) -> RuntimeCycleResult:
        start = time.monotonic()
        result = RuntimeCycleResult(signal=signal, context_kernel=kernel)

        # 1. Perceive
        percept = self._perceptor.perceive(signal, kernel)
        result.percept = percept

        # 2. Build working graph
        uol_graph = self._graph_builder.build(percept)
        result.uol_graph = uol_graph

        # 3. Attend (select focus)
        working_set = self._attention.attend(uol_graph, kernel, kernel.budget)
        result.working_set = working_set

        # 4. Retrieve (graph-focused)
        retrieval = self._retrieval_executor.retrieve(uol_graph, working_set, kernel)
        result.retrieval = retrieval

        # 5. Resolve (concepts, ports, affordances)
        resolution = self._resolver.resolve(uol_graph, working_set, kernel)
        result.resolution = resolution

        # 6. Plan
        act_plan = self._planner.plan(
            conversation_act=percept.conversation_act,
            situation=percept.situation,
            safety_frame=percept.safety_frame,
            fact_candidates=percept.fact_candidates,
            meaning_percept=percept,
        )
        result.act_plan = act_plan

        # 7. Extract patches
        patches = self._patch_extractor.extract(uol_graph, act_plan)
        result.patch_candidates = patches

        # 8. Validate patches
        for patch in patches:
            validation = self._patch_validator.validate(patch, kernel)
            result.validation.append(validation)
            if validation.status == "accepted":
                self._apply_patch(patch, kernel)

        # 9. Consolidate (async)
        for patch in result.validation:
            if patch.status == "accepted":
                self._consolidator.consolidate(patch)

        result.cost_ms = (time.monotonic() - start) * 1000
        return result
```
