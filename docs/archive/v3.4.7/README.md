# Archived CEMM v3.4.7 contracts

Files in this directory are historical records of the v3.4.7 semantic-authority design.

They are **not active implementation guidance** and must not be referenced by canonical runtime code, current tests, or new design work except for migration analysis.

The active authority order is defined by the repository root `AGENTS.md`.

The documentation-cutover installer copies the repository's existing v3.4.7 contract files here before replacing or deleting them.
